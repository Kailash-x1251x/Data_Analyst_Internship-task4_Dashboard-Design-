# Task 4: Executive Sales Dashboard

## 🚀 Objective
The goal of this task was to design an interactive, executive-level dashboard using **Tableau**. The dashboard is intended for high-level stakeholders to monitor business health (Sales, Profit, Quantity) at a glance while allowing for deep dives into regional performance.

## 🛠️ Tools Used
- **Tableau Public** (Dashboard Design & Interactivity)
- **Dataset:** Sample - Superstore

## 📊 Dashboard Features

### 1. KPI Cards (The "Heads-Up" Display)
- **Design Choice:** Placed at the top for immediate visibility.
- **Metrics:**
  - **Total Sales:** $2.3M (Monitors revenue).
  - **Total Profit:** $286k (Monitors health).
  - **Total Quantity:** 37k (Monitors volume).

### 2. Interactivity & User Experience
- **Action Filters:** The dashboard features a **Geography Filter**. Clicking on a specific state (e.g., California) on the map instantly updates the KPI cards and Segment charts to show data for *only* that region.
- **Benefit:** Allows regional managers to generate their own specific reports without needing multiple dashboards.

### 3. Visual Breakdown
- **Profit Map:** Uses a diverging color palette (Orange to Blue) to instantly highlight profitable vs. unprofitable states.
- **Segment Analysis (Pie Chart):** visualizing the contribution of Consumer, Corporate, and Home Office segments to total revenue.

## 📉 Final Dashboard
The dashboard provides a "single pane of glass" view for decision-makers.

---
### 📄 Presentation Summary
*A summary presentation deck was prepared to accompany this dashboard. Key recommendations included:*
1. **Invest in the West:** The West region shows the highest profit margins.
2. **Review Pricing:** Certain high-volume states are showing negative profit (orange on map), indicating pricing inefficiencies.
